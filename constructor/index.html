Constructor

What is a constructor

The constructor method is a method that is used to create and initialize an object when using the class keyword. There can only be one method that uses the constructor keyword within a class. A SyntaxError exception will be thrown if the class contains more than one constructor method. The constructor can use the super keyword to call the constructor of the parent class

How constructor works

  • To avoid repetition let's say we want to create many objects, let's say this object are persons it will be very restrictive if we have to create a lot of persons and we will not respect the DRY. So we can create a constructor who will be like a reference where all object will inherit of his property and methods.
  • To create a constructor we only create a constructor() function.
  • By adding parameter to the function with the this keyword who point on this parameter we indicated that the parameter create news property.
  • Like we know the this keyword point from the object where he's called, so in our example the this keyword point the the window object. To avoid that when we create a new object we simply add the new keyword before the constructor function to create a new empty object and declare that the parameters are the new property of our object.
