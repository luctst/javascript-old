Const, let

  • With es6 you can declares variables with two new terms let, const.
  • With const you can store variables like we did with var except that you can't change the values of a const.
  • let works like var but there a some differencies.
  • var and const create a block-scope which means that if you create many let, const in a diferent scope it will create many variables.

Immediatly invoked functions expression IIFE.

  • Remember IIFE is useful when you want call a function only once and when you want create a unique scop.
  • In es6 we know that {} create a scope combine with let or const so you can simply create an IIFE by using {}.

Template literal strings in es6

  • To display strings with es6 you must use backtips `` and ${nameOfYourVariable} to display the content.

Destructuring

  • Destructuring is helpful when you want extract values from an array or object .
  • There are many possibilities of destructuring, for an array start to declare your const, let then open [] which indicates that you extract values from an array, after that you just need to declare the new name of your values, finally use the = and declare your array where the values will be extracts.
  • For object there are two ways of using destructuring, the first one is like array but instead of using []you have to use {} and write the exact same properties, and the second one is a little bit different because you can create new variables simply add propertiesObject: nameOfYourVariable and that's it you create a new variable with the value of the property of the object.

Loops

  • Es6 include two new ways of declaring functions for of and for in .
  • The first new method is for of with this method you can simply declare your variable and after the of keyword include the parameter that you want loop over. this method return the value inside your parameter .
  • The second method is for in with this one who works exactly the same than for of but instead to return value this one will return the index position of the elements in your parameter .
