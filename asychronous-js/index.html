Asynchronous javascript

What is asynchronous javascript

Synchronous javascript means that your code is executed line by line, this involves that you have to wait that the line above ending before execute the others lines. Asynchronous javascript is the opposite of synchronous javascript, which means that you don't have to wait a line of code to be finish to start an other one.
