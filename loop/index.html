For loop

  • For loop is one of the most used method in loops, the for loop has 3 steps to work.
  • A loop in javascript work with three steps, the first one is the initialisation during this phase we declare our iterator generally i, the second phase is the condition for this phase add a condition in the loop, finally the last step is the iteration in this phase we iterate our initiator. Let's see after how the steps works.
  • The first step is the initialisation during this phase we gonna create a variable generally i and we gonna give a value to this variable.
  • the second phae is condition for this phase you gonna create a condition to loop, it can be a number like in our first example but also the lenght of an array, object.. you can add anything.
  • The third phase is iteration for this phase you gonna add or substract your iterator it can be by 1, 100 or anything else.
  • The process of a loop is this one, first the loop check the value of your iterator, then check if the iterator filled the conditionif not the loop stop.
