This keyword in javascript

What is the this keyword

In most cases, the value of this will be determined from the way a function is called. It is not possible to assign a value to it at run time and its value may be different each time the function is called.

This keyword in practice

  • The this keyword is a little bit different, he returns the value of the context where it is called, let's try to explain with our examples above.
  • In our first example we create a variable keyword who is equal to this and then we call our keyword variable in the global context in other terms in the window context so the window object is return.
  • The this keyword acts diferent when he's called in a different scope like function or object.
