Scope in javascript

Understand how works scope

A scope in programming language is an environment where we can access variables, functions.. In Javascript if, for, while.. don't create a scope. So in a script the global scope define all variables written directly on the document root, in our example A variable is in the global scope. The scope only have acess to variables define inside his own scope, for example we can't access to the b variable in the global scope because she's in the first function scope. But we can acess to a 'parent' scope, for example we can access inside the first function to the A variable which is define in the global scope.

Example with scope

  • We create a A variable in the global scope.
  • We call first() function which is declare in the global scope.
  • We create a new scope by creating the first() function.
  • We add in the first() function scope a B variable which is in the first() scope.
  • We call the second() function which is declare in the first() scope.
  • We create a new scope by creating the second() function.
  • We add in the second() function scope a C variables which is define in the second() scope.
  • We call the third() function who is available because she's in the parent global scope.
  • Back in the global scope we create a new scope who is a function third().
  • We create in the third() scope a D variable which is define in the third() scope.
  • In this example we have an error because we try to call a variable who is C in the third() scope but this variable is define in the second() scope so it's impossible, because we know that we can only access variable, functions... in the parent scope.

The solution !!
