What is array ?

An array in programmation is list or a collection of datas(primitives or objects).They are very often use for store many values in one variable, This is compared to a variable that can only store one value.

Source by developer.mozilla.org

  2. So first we declare our variable with a good name who will represent the datas that we stores in our example names.

Methods and properties

Here are some famous properties and method with arrays.

  1. First we create an array.
  2. Like all objects in javascript they have a list of method and properties that we can use on our arrays.
  3. length return the lenght of the arrays, very useful in loop.
  4. push() add at the end of the array the parameter in ().
  5. unshift() add as first element in the array the parameter in ().
  6. pop() delete the last element in our array.
  7. shift() delete the first element in our array.
